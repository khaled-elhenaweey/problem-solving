<h2><a href=https://www.codewars.com/kata/58f8a3a27a5c28d92e000144/train/javascript target="_blank">Find the first non-consecutive number</a></h2><h3>8 kyu</h3><p data--h-bstatus="0OBSERVED">Your task is to find the first element of an array that is not consecutive.</p><p data--h-bstatus="0OBSERVED">By not consecutive we mean not exactly 1 larger than the previous element of the array.</p><p data--h-bstatus="0OBSERVED">E.g. If we have an array <code data--h-bstatus="0OBSERVED">[1,2,3,4,6,7,8]</code> then <code data--h-bstatus="0OBSERVED">1</code> then <code data--h-bstatus="0OBSERVED">2</code> then <code data--h-bstatus="0OBSERVED">3</code> then <code data--h-bstatus="0OBSERVED">4</code> are all consecutive but <code data--h-bstatus="0OBSERVED">6</code> is not, so that's the first non-consecutive number.</p><p data--h-bstatus="0OBSERVED">If the whole array is consecutive then return <code data--h-bstatus="0OBSERVED">null</code><sup data--h-bstatus="0OBSERVED">2</sup>.</p><p data--h-bstatus="0OBSERVED">The array will always have at least <code data--h-bstatus="0OBSERVED">2</code> elements<sup data--h-bstatus="0OBSERVED">1</sup> and all elements will be numbers. The numbers will also all be unique and in ascending order. The numbers could be positive or negative and the first non-consecutive could be either too!</p><hr data--h-bstatus="0OBSERVED"><p data--h-bstatus="0OBSERVED"><sup data--h-bstatus="0OBSERVED">1</sup> Can you write a solution that will return <code data--h-bstatus="0OBSERVED">null</code><sup data--h-bstatus="0OBSERVED">2</sup> for both <code data--h-bstatus="0OBSERVED">[]</code> and <code data--h-bstatus="0OBSERVED">[ x ]</code> though?  (This is an empty array and one with a single number and is not tested for, but you can write your own example test. )</p><p data--h-bstatus="0OBSERVED"><sup data--h-bstatus="0OBSERVED">2</sup>  </p><ul data--h-bstatus="0OBSERVED"><li data--h-bstatus="0OBSERVED">Swift, Ruby and Crystal: <code data--h-bstatus="0OBSERVED">nil</code></li><li data--h-bstatus="0OBSERVED">Haskell: <code data--h-bstatus="0OBSERVED">Nothing</code>  </li><li data--h-bstatus="0OBSERVED">Python, Rust, Scala, Lambda Calculus: <code data--h-bstatus="0OBSERVED">None</code>  </li><li data--h-bstatus="0OBSERVED">Julia: <code data--h-bstatus="0OBSERVED">nothing</code>  </li><li data--h-bstatus="0OBSERVED">Nim: <code data--h-bstatus="0OBSERVED">none(int)</code> (See <a href="https://nim-lang.org/docs/options.html" data-turbolinks="false" target="_blank" data--h-bstatus="0OBSERVED">options</a>)</li><li data--h-bstatus="0OBSERVED">C++: <code data--h-bstatus="0OBSERVED">std::nullopt</code></li></ul>
